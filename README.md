# IT(AMHS) Job Tutorial Web Demo

Interactive Korean web demo for understanding semiconductor FAB AMHS roles, bottlenecks, KPIs, and candidate fit.

Live site: https://uhchoi2132.github.io/

## Contents

- AMHS job analysis for new-hire training
- Interactive FAB logistics bottleneck simulator
- KPI dashboard for transport time, queue, equipment idle, deadlock risk, and throughput
- Candidate fit analysis based on autonomous systems, optimization, and simulation background
- Operational AMHS troubleshooting playbook
